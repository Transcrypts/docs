## Overview

The Bank Accounts endpoints manage bank account information for Direct Deposit Switching (DDS) and Paycheck Linked Loans (PLL) products. These endpoints allow you to access information from bank accounts connected to a user's payroll account during income and employment verification before making deposit switches.

## Attributes

Bank account objects contain the following attributes:

| Attribute | Type | Description |
|-----------|------|-------------|
| `account_number` | string | Bank account number |
| `routing_number` | string | Routing number |
| `account_name` | string | User-friendly account name for bank account |
| `account_type` | string | Account type: **C** (Checking) or **S** (Savings) |
| `deposit_type` | string | Deposit type: **E** (Entire paycheck), **P** (Percentage), or **A** (Fixed amount) |
| `deposit_value` | string | Deposit value of selected bank account |
| `bank_name` | string | Bank name |

## Endpoints

The available endpoint for bank account management:

- List all bank accounts
