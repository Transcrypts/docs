---
title: "Retrieve a Deposit Switch Report"
description: "Retrieves a deposit switch report for a user"
api: "GET /v1/users/{user_id}/deposit_switch/report/"
---

## Overview

This endpoint retrieves a deposit switch report for a user in the Transcrypts system.

## Authentication

This endpoint requires authentication credentials:
- **ClientID** (Header)
- **AccessKey** (Header)

## Path Parameters

<ParamField path="user_id" type="string" required>
  The unique identifier for the user whose deposit switch report is being retrieved.
</ParamField>

## Response

The endpoint returns a deposit switch report containing information about the user's direct deposit switching status and details.

### Error Responses

| Status Code | Description |
|-------------|-------------|
| **401** | Unauthorized - Authentication credentials are invalid or missing |
| **403** | Forbidden - Insufficient permissions |
| **404** | Not Found - The specified user was not found |

<Note>
  The user must have completed the deposit switch process through the Bridge before a report can be retrieved.
</Note>
