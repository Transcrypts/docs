The Transcrypts JavaScript/TypeScript SDK (`transcrypts`) provides a client-side interface for launching the Transcrypts employment verification widget, handling verification results, and subscribing to widget lifecycle events.

This reference covers every public class, method, interface, and type exported by the SDK. For quickstart guides and tutorials, see the [Getting Started](/getting-started/quickstart) guide.

## Package Details

| Field       | Value                                            |
|-------------|--------------------------------------------------|
| npm package | `transcrypts`                                    |
| Version     | `0.1.0`                                          |
| License     | MIT                                              |
| Module      | ESM (`dist/index.js`) and CJS (`dist/index.cjs`) |
| Types       | `dist/index.d.ts`                                |

## Exports

The SDK exports the following public symbols from its entry point:

| Export                 | Kind      |
|------------------------|-----------|
| `Transcrypts`          | Class     |
| `TranscryptsOptions`   | Interface |
| `VerifyOptions`        | Interface |
| `VerificationResult`   | Interface |
| `WorkExperience`       | Interface |
| `TranscryptsEventType` | Type      |
| `TranscryptsEvent`     | Interface |

## Installation

```bash
npm install transcrypts
```
