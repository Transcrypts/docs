## Description

This endpoint establishes a customization template for your application needs. The customization template feature enables branding and configuration adjustments for the Bridge widget integration.

## Authentication

Requires API credentials passed via headers:
- Client ID (header)
- Access Key (header)

## Request Body

The endpoint accepts JSON configuration for template customization properties including bridge settings, branding, orders configuration, and document upload specifications.

## Response

### Success Response (201 Created)

The API returns template configuration details upon successful creation.

### Error Responses

**400 Bad Request:** Returned when request parameters are incorrect or invalid.

**401 Unauthorized:** Returned when authentication credentials are missing or invalid.

```json
{
  "error": {
    "code": "authentication_failed",
    "message": "No such token"
  }
}
```

**403 Forbidden:** Returned when authentication credentials are not provided.

```json
{
  "error": {
    "code": "not_authenticated",
    "message": "Authentication credentials were not provided."
  }
}
```
