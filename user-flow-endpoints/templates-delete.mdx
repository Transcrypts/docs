---
title: "Delete a Customization Template"
description: "Remove a customization template from the system"
api: "DELETE /v1/templates/{template_id}"
---

## Description

This endpoint removes a customization template from the system. The operation destroys the specified template resource identified by its unique ID.

## Authentication

The endpoint requires API credentials passed via HTTP headers:
- Client ID (header)
- Access Key (header)

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `template_id` | string | Yes | The unique identifier of the customization template to remove |

## Response

### Success Response (204 No Content)

The successful deletion returns an empty response body with a 204 status code, confirming the template has been removed.

### Error Responses

**401 Unauthorized:**

```json
{
  "error": {
    "code": "authentication_failed",
    "message": "No such token"
  }
}
```

**403 Forbidden:**

```json
{
  "error": {
    "code": "not_authenticated",
    "message": "Authentication credentials were not provided."
  }
}
```

**404 Not Found:**

```json
{
  "detail": "Not Found."
}
```
