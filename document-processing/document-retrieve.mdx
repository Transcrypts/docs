# Retrieve Document

This endpoint returns a specific document associated with a link.

## Authentication

This endpoint requires authentication using:
- **ClientID** (header)
- **AccessKey** (header)

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `link_id` | string | Link identifier (required) |
| `doc_id` | string | Document identifier (required) |

## Response

The endpoint returns the document data in JSON format for the specified document within the given link.

## Error Responses

### 401 Unauthorized

```json
{
  "error": {
    "code": "authentication_failed",
    "message": "No such token"
  }
}
```

### 403 Forbidden

```json
{
  "error": {
    "code": "not_authenticated",
    "message": "Authentication credentials were not provided."
  }
}
```

### 404 Not Found

```json
{
  "detail": "Not Found."
}
```
