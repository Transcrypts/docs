# Finalize a Collection

This endpoint processes a document collection by establishing or refreshing connections for all matched documents and users. It accepts an optional subset of document identifiers for selective finalization; if none are specified, all recognized documents get processed.

## Authentication

This endpoint requires API credentials via header authentication:
- **ClientID** (header)
- **AccessKey** (header)

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `collection_id` | string | The unique identifier for the collection being finalized (required) |

## Request Body

The endpoint accepts optional document filtering:

```json
{
  "document_ids": ["d0c1234567890abcdef1234567890abc"]
}
```

- **document_ids** (array, optional): Array of specific document identifiers to finalize. If omitted, the system processes all documents within the collection.

## Response

The endpoint returns finalization status and results, including:
- Processing status indicators
- Created or updated link information
- User and document associations
- Any validation or processing errors encountered

## Error Responses

| Status | Description |
|--------|-------------|
| `400` | Invalid request parameters |
| `401` | Authentication failure |
| `403` | Insufficient permissions |
| `404` | Collection not found |

### 401 Unauthorized

```json
{
  "error": {
    "code": "authentication_failed",
    "message": "No such token"
  }
}
```

### 403 Forbidden

```json
{
  "error": {
    "code": "not_authenticated",
    "message": "Authentication credentials were not provided."
  }

```
