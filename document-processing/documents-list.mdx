# List All Documents

This endpoint retrieves a paginated collection of documents associated with a specific link in the Transcrypts system. Documents uploaded through the Bridge can be accessed via this GET request.

## Authentication

This endpoint requires authentication credentials:
- **ClientID** (header)
- **AccessKey** (header)

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `link_id` | string | The unique identifier for the link whose documents you want to retrieve (required) |

## Response

The endpoint returns document data in JSON format containing the list of documents associated with the specified link.

## Error Responses

```json
{
  "error": {

  }
}
```

### 403 Forbidden

```json
{

```

### 404 Not Found

```json

```
