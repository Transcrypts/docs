# List All Document Collections

Returns a paginated list of document collections for the authenticated user.

## Authentication

This endpoint requires API credentials:
- **ClientID** (header)
- **AccessKey** (header)

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `page` | integer | No | A page number within the paginated result set |
| `page_size` | integer | No | Number of results to return per page |

## Response (200 OK)

```json
{
  "count": 2,
  "next": "https://prod.transcrypts.com/v1/documents/collections/?page=2",
  "previous": null,
  "results": [
    {
      "collection_id": "a1b2c3d4e5f6478899aabbccddeeff00",
      "created_at": "2025-11-11T10:00:00Z",
      "updated_at": "2025-11-11T10:05:00Z",
      "files_count": 3,
      "documents_count": 5,
      "users_count": 2
    }
  ]
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `count` | integer | Total number of collections |
| `next` | string (URI) | Link to next page, or null if final page |
| `previous` | string (URI) | Link to previous page, or null if first page |
| `results` | array | Array of collection objects |

### Collection Object Fields

| Field | Type | Description |
|-------|------|-------------|
| `collection_id` | string | Unique collection identifier |
| `created_at` | string (date-time) | Collection creation timestamp |
| `updated_at` | string (date-time) | Last update timestamp |
| `files_count` | integer | Number of uploaded files |
| `documents_count` | integer | Number of recognized documents |
| `users_count` | integer | Number of users in collection |

## Error Responses

### 401 Unauthorized

```json
{
  "error": {
    "code": "authentication_failed",
    "message": "No such token"
  }
}
```

### 403 Forbidden

```json
{
  "error": {
    "code": "not_authenticated",
    "message": "Authentication credentials were not provided."
  }
}
```
