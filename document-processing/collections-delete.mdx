# Delete a Document Collection

This endpoint performs a soft-delete operation on a document collection and its associated files. The collection and files are marked as deleted but remain stored in the system rather than being physically removed.

## Authentication

This endpoint requires API credentials:
- **ClientID** (header)
- **AccessKey** (header)

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `collection_id` | string | The unique identifier for the collection to delete |

## Response

### 204 No Content

Successful deletion. No response body is returned.

## Error Responses

### 401 Unauthorized

```json
{
  "error": {
    "code": "authentication_failed",
    "message": "No such token"
  }
}
```

### 403 Forbidden

```json
{
  "error": {
    "code": "not_authenticated",
    "message": "Authentication credentials were not provided."
  }
}
```

### 404 Not Found

```json
{
  "detail": "Not Found."
}
```
