# Get Document Collection Details

This endpoint retrieves comprehensive information about a specific document collection, including all uploaded files and categorized documents.

## Authentication

This endpoint requires API credentials:
- **ClientID** (header)
- **AccessKey** (header)

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `collection_id` | string | Collection ID (required) |

## Response Fields

### Collection Metadata
- **collection_id**: Unique identifier for the collection
- **created_at**: Timestamp of collection creation
- **updated_at**: Timestamp of last update

### Uploaded Files Array

Each file contains:
- **file_id**: Unique file identifier
- **filename**: Original filename
- **mime_type**: File format (PDF, JPEG, PNG, TIFF, WebP, BMP, HEIC, HEIF)
- **status**: Processing status (`pending`, `validating`, `validated`, `invalid`, `duplicate`, `processing`, `successful`, `failed`)
- **validations**: Object containing file validation checks
  - `is_viable_size`
  - `is_supported_type`
  - `is_accessible`
  - `is_valid`
  - `is_readable`
  - `is_unique`
- **user_id**: Associated Transcrypts user ID
- **external_user_id**: Associated external system ID

### Documents Array

Each recognized document contains:
- **document_id**: Unique document identifier
- **file_id**: Associated file ID
- **document_type**: Categorized type (`PAYSTUB`, `W2`, `F1099`, `F1040`, `BANK_STATEMENT`, `PASSPORT`, `GREEN_CARD`, `DRIVER_LICENSE`, `LETTER_OF_VERIFICATION`, `UTILITY_BILL`, `LEASE_AGREEMENT`, `INSURANCE_HOME_POLICY`, `INSURANCE_AUTO_POLICY`, `VOLUNTEER_LETTER`, `OTHER`)
- **document_subtype**: Optional subtype for tax documents
- **status**: Document processing status
- **first_name** / **last_name**: Extracted names
- **user_id** / **external_user_id**: User associations
- **start_page** / **end_page**: Document page range

### Users Array

Each user entry includes:
- **id**: Transcrypts user identifier
- **external_user_id**: External system identifier
- **first_name**: User's first name
- **last_name**: User's last name
