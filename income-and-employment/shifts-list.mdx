## Description

This endpoint retrieves a comprehensive list of work shifts associated with a specific link. The response includes ongoing and completed work shifts, time entries (clock in/clock out), the type of work, and earnings.

## Authentication

This endpoint requires API credentials:
- **ClientID** (header)
- **AccessKey** (header)

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `link_id` | string | Yes | The unique identifier for the link |

## Response

The endpoint returns shift data in JSON format including:
- Clock in/out timestamps
- Work type classifications
- Associated earnings data
- Shift completion status (ongoing vs. completed)
