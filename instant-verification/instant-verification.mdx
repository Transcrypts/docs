## Overview

Instant Verification objects represent a real-time verification request where a verifier submits a subject's name, date of birth, and address history to receive employment and income results. Unlike link-based verification, Instant Verification does not require the end user to connect a payroll account.

## Attributes

| Attribute | Type | Description |
|-----------|------|-------------|
| id | string | Unique identifier for the verification request |
| status | string | Current status: `pending`, `processing`, `completed`, `failed`, `partial` |
| created_at | datetime | ISO 8601 timestamp of when the request was created |
| completed_at | datetime | ISO 8601 timestamp of when the verification completed (null if not yet completed) |
| subject | object | The person being verified |
| addresses | array | Address history for the subject |
| results | object | Verification results (populated when status is `completed` or `partial`) |
| tracking_info | string | Optional client-side tracking reference |

## Subject Object

| Attribute | Type | Description |
|-----------|------|-------------|
| first_name | string | First name of the subject |
| last_name | string | Last name of the subject |
| date_of_birth | date | Date of birth in YYYY-MM-DD format |
| ssn | string | Social Security Number, last 4 digits or full (optional) |

## Address Object

| Attribute | Type | Description |
|-----------|------|-------------|
| street | string | Street address |
| city | string | City |
| state | string | State |
| zip | string | Zip code |
| country | string | Country (default: "US") |
| from_date | date | Date the subject began residing at this address |
| to_date | date | Date the subject left this address (null if current) |

## Results Object

| Attribute | Type | Description |
|-----------|------|-------------|
| employments | array | Employment records found for the subject |
| income_summary | object | Summary of income data across matched employments |
| confidence_score | number | Confidence in the match, ranging from 0 to 1 |
| sources | array of strings | Data sources used to compile the results |
