## Overview

This page documents how to retrieve and understand transaction data from financial accounts connected through the Transcrypts API.

## Attributes

The transactions response includes the following top-level attributes:

| Attribute | Type | Description |
|-----------|------|-------------|
| count | integer | Total number of items |
| next | uri | Link to next page of results |
| previous | uri | Link to previous page of results |
| accounts | array of objects | List of account objects |
| transactions | array of objects | List of transaction objects |

## Accounts Object

Account objects contain the following fields:

| Attribute | Type | Description |
|-----------|------|-------------|
| id | string | Unique account identifier |
| created_at | date-time | Account creation timestamp |
| updated_at | date-time | Most recent update timestamp |
| type | string | Parent account type |
| subtype | string | Account subtype classification |
| mask | string | Masked account number |
| nickname | string | Custom account name |
| balances | array of objects | Balance information array |

### Balances Object

| Attribute | Type | Description |
|-----------|------|-------------|
| currency_code | string | ISO currency designation |
| balance | string | Current balance amount |
| available_balance | string | Available funds amount |
| credit_limit | string | Maximum credit allowance |

## Transactions Object

Transaction objects include these fields:

| Attribute | Type | Description |
|-----------|------|-------------|
| id | string | Unique transaction identifier |
| created_at | date-time | Transcrypts creation timestamp |
| updated_at | date-time | Last modification timestamp |
| account_id | string | Associated account identifier |
| external_id | string | External system transaction ID |
| amount | string | Monetary transaction value |
| currency_code | string | ISO 4217 currency code |
| check_number | string | Check identifier if applicable |
| categories | array of strings | Assigned transaction categories |
| description | string | Human-readable transaction summary |
| status | string | Either "POSTED" or "PENDING" |
| type | string | Either "CREDIT" or "DEBIT" |
| posted_at | date-time | Funds movement timestamp |
| transacted_at | date-time | Transaction occurrence timestamp |
| memo | string | Additional transaction details |
| merchant_category_code | string | ISO 18245 transaction code |
| location | object | Geographic transaction data |

### Location Object

| Attribute | Type | Description |
|-----------|------|-------------|
| latitude | float | Geographic latitude coordinate |
| longitude | float | Geographic longitude coordinate |

## Endpoints

**Available endpoint:**
- List all bank transactions - retrieves transaction data for connected accounts
