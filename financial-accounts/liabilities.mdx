## Endpoint Overview

This endpoint retrieves liability account information associated with a user's financial profile. The response includes details about credit cards, loans, and other debt obligations with associated balance and payment information.

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| link_id | string | Yes | The unique identifier for the user's account link |

### Authentication

Requires API credentials:
- ClientID
- AccessKey

## Response Structure

The endpoint returns liability accounts with comprehensive financial details including:
- Account types (credit cards, loans, etc.)
- Current balances
- Payment information
- Account status details

## Implementation Notes

- Authentication credentials must be provided via header
- The link_id is required in the endpoint path

## Related Endpoints

This endpoint is part of the Financial Accounts section and operates alongside:
- Bank account retrieval
- Transaction listing
- Income report generation
- Balance inquiries
