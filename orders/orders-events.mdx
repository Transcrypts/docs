## Endpoint Overview

**GET** `https://prod.transcrypts.com/v1/orders/{id}/events/`

Returns a list of events for the specified order.

## Authentication

This endpoint requires API credentials:
- **ClientID** (Header)
- **AccessKey** (Header)

## Parameters

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | **Required.** The order identifier |

## Response

The endpoint returns event data associated with a particular order, providing a history of actions and status changes.
