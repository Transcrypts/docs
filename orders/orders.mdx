## Overview

An **Order** represents a customized landing page created for users to connect their accounts. These personalized invitations can be sent via email, text message, or both. Orders are customizable across three key components:

- Landing page
- Text message
- Email

Users access their personalized landing pages through invitation links, and Orders can be created either through the Transcrypts Dashboard or programmatically using API endpoints.

## Core Attributes

| Attribute | Type | Description |
|-----------|------|-------------|
| `id` | string | Unique order identifier |
| `products` | array | Selected products for the order |
| `source` | string | Platform type: `internal`, `accio`, `tazworks`, `encompass`, `simplenexus`, `external_webpage` |
| `order_number` | string | External client ID |
| `custom_field` | string | Client-provided field for email, SMS, or landing page (dashboard-enabled only) |
| `client_name` | string | Verification agency name |
| `first_name` | string | User first name |
| `last_name` | string | User last name |
| `user_id` | string | Unique user identifier |
| `share_url` | string | Landing page URL for sharing |
| `created_at` | string | ISO 8601 creation timestamp |
| `updated_at` | string | ISO 8601 last update timestamp |
| `canceled_at` | string | ISO 8601 cancellation timestamp |
| `expired_at` | string | ISO 8601 expiration timestamp |
| `is_expired` | bool | Expiration status |
| `initial_order` | string | Origin order ID from refresh action |
| `refresh_order` | string | Latest refresh order ID |
| `employers` | array | User employer list (see Employers object) |
| `insurance` | array | Insurance metadata (see Insurance object) |
| `bank` | array | Bank transaction metadata (see Bank object) |
| `manager` | array | Order manager info (see Manager object) |
| `loan` | array | Loan information (see Loan object) |
| `template_id` | string | Template identifier |
| `voie_report_id` | string | Income/employment report ID (Fannie Mae DU, Freddie Mac LPA compatible) |
| `voa_report_id` | string | Asset verification report ID |
| `income_insights_report_id` | string | Bank income insights report ID |
| `aim_check_report_id` | string | AIM check report ID for income from uploaded documents |
